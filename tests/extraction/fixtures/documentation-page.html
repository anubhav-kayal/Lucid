Getting Started with React

React is a JavaScript library for building user interfaces. This page will help you get started with React in your project.

Quick Start

The fastest way to start using React is to include it from a CDN. However, for production applications, we recommend using a build tool like Vite or Next.js.

npx create-react-app my-app
cd my-app
npm start

Your First Component

React applications are built from components. A component is a JavaScript function that returns markup:

function Welcome() {
  return <h1>Hello, World!</h1>;
}

Using JSX

JSX is a syntax extension for JavaScript that allows you to write HTML-like markup inside your JavaScript code. JSX is not required to use React, but it makes the code more readable.

Hooks

Hooks let you use state and other React features without writing a class. The most commonly used hooks are:

  • useState — manage component state
  • useEffect — perform side effects
  • useContext — access context values
  • useRef — reference DOM elements

Next Steps
